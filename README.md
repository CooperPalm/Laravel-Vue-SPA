## Features

-   Laravel 8
-   Vue + VueRouter + Vuex + VueI18n + ESlint
-   Pages with dynamic import and custom layouts
-   Login, register, email verification and password reset
-   Authentication with JWT
-   Socialite integration
-   Bootstrap 5 + Font Awesome 5
